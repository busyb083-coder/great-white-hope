### **Part 2: Deploy Backend to Railway (10 minutes)**

1. Go to https://railway.app
2. Click "New Project"
3. Select "Deploy from GitHub repo"
4. Select `great-white-hope` repository
5. **Root Directory**: Set to `apps/backend`
6. **Build Command**: `npm run build`
7. **Start Command**: `node dist/index.js`
8. **Environment Variables**:
   ```
   NODE_ENV=production
   PORT=3001
   DATABASE_URL=postgresql://[auto-generated by Railway]
   JWT_SECRET=your-super-secret-key-change-this
   CORS_ORIGIN=https://your-vercel-frontend.vercel.app
   API_BASE_URL=https://your-railway-backend.up.railway.app
   ```
9. Add PostgreSQL database (Railway â†’ New â†’ Database â†’ PostgreSQL)
10. Copy PostgreSQL connection string to `DATABASE_URL`
11. Click "Deploy"
12. âœ… Wait 3-5 minutes for deployment
13. Get your Railway URL: `https://great-white-hope-production.up.railway.app`
